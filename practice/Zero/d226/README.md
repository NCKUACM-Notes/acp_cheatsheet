# ZOJ-d226 - Back to High School Physics

某一個粒子有一初速度和等加速度。假設在 t 秒後此粒子的速度為 v ，請問這個粒子在 2t 秒後所經過的位移是多少。

> * 題目來源：**UVA-10071**
> * 記憶體限制：512 MB
> * 公開測資點#0 (100%): 1.0s , < 1M

---
## Input

每組測試資料1列，有2個整數，分別代表 v (-100 <= v <=100) 和 t (0 <= t <= 200)。

---
## Output

對每組測試資料請輸出這個粒子在 2t 秒後所經過的位移是多少。

---
## Sample Input

```
0 0
5 12 
```

---
## Sample Output

```
0
120
```
